Visão Geral dos Wireframes

Estes wireframes representam a interface do sistema SIG-Aldeias, focando-se na usabilidade operacional para a Proteção Civil. O design segue princípios de:

  • Mobile-first: Interface responsiva para dispositivos móveis
  • Identidade institucional: Cores da Proteção Civil (azul escuro)
  • Operacionalidade: Acesso rápido a informações críticas
  • Decision Support: Indicadores visuais de prioridade

🎯 Objetivo

Transformar dados territoriais em apoio à decisão operacional durante emergências.

📊 Prioridades

Cores indicam nível de risco: Crítico, Médio, Baixo
