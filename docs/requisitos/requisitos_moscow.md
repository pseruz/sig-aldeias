# Requisitos SIG-Aldeias

## Must have (Obrigatório)

### Funcionais
- RF1: Registo de agregados familiares com coordenadas GPS e atributos estruturais
- RF2: Validação de registos por técnicos da Proteção Civil
- RF3: Visualização geográfica interativa com marcadores
- RF4: Filtros por vulnerabilidade (idosos, crianças) e material estrutural
- RF5: Autenticação de utilizadores com controlo de acesso por perfil

### Não Funcionais
- RNF1: Sistema acessível via browser (desktop e mobile)
- RNF2: Tempo de carregamento do mapa < 3 segundos
- RNF3: Conformidade com GDPR para dados sensíveis

## Should have (Importante)
- RF6: Exportação de relatórios em CSV e PDF
- RF7: Representação 2.5D de edifícios
- RF8: Logs de acesso e operações

## Could have (Desejável)
- RF9: Notificações push para atualizações críticas
- RF10: Integração com sistemas GIS existentes

## Won't have (Fora de âmbito)
- RF11: Aplicação móvel nativa (Android/iOS)
- RF12: Sincronização offline
- RF13: Integração automática com BIM/Revit
